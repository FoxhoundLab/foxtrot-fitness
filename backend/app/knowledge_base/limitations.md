# Limitations → Movement Substitutions

When a user reports a limitation, this file maps it to specific movements that
should be EXCLUDED from their program and what to substitute instead.

The AI must check user limitations against this map before finalizing a program.

---

## Knee Injury / Knee Pain

**Avoid these movements:**
- Back squat (full depth)
- Front squat
- Bulgarian split squat (deep)
- Walking lunges (deep)
- Reverse lunges (deep)
- Box jumps
- Plyo box jumps
- Jump rope (high impact)
- Running (high impact)
- Leg press (deep with heavy load)
- Leg extension (heavy)
- Hack squat (heavy)
- Pendulum squat

**Substitute with:**
- Goblet squat (lighter weight, controlled depth)
- Belt squat (minimal spinal loading)
- Leg press (feet high, moderate weight)
- Step-ups (lower box, controlled tempo)
- Incline treadmill walking (low impact cardio)
- Cycling (low impact)
- Swimming (if available)
- Elliptical (low impact)
- Leg extension (light, controlled)
- Hamstring curls, hip thrusts, glute bridges (knee-friendly posterior chain)
- Seated calf raises (less knee stress than standing)

**Modified card options:**
- Replace "high-impact cardio" with low-impact alternatives
- Reduce squat depth to pain-free range
- Use tempo control (3-1-2) to reduce momentum

## Shoulder Injury / Shoulder Pain (Rotator Cuff, Impingement)

**Avoid these movements:**
- Overhead press (barbell, heavy)
- Behind-the-neck lat pulldown
- Behind-the-neck press
- Upright rows (narrow grip)
- Dips (deep, weighted)
- Bench press (flat, heavy)
- Incline bench press (heavy)
- Handstand push-ups
- Ring dips (deep)
- Lateral raises (above shoulder height)
- Front raises (heavy)

**Substitute with:**
- Neutral grip dumbbell press
- Floor press (limited ROM)
- Landmine press (scapular-friendly)
- Cable chest fly (controlled, light)
- Pec deck (supported)
- Seated chest press machine
- Machine shoulder press (guided path)
- Cable lateral raises (lower height)
- Face pulls
- Band pull-aparts
- External rotations with band/cable
- Incline treadmill walking
- Cycling (upright, neutral grip)
- Rower (lighter resistance)

**Modified options:**
- Keep all pressing below shoulder height
- Prioritize pulling over pressing
- Reduce pressing volume by 50%

---

## Lower Back Injury / Lower Back Pain (Disc, Sciatica, etc.)

**Avoid these movements:**
- Conventional deadlift (heavy)
- Stiff-leg deadlift (heavy)
- Good mornings
- Bent-over rows (heavy)
- Back extensions (weighted)
- GHD extensions (heavy)
- Reverse hypers (heavy)
- Barbell overhead squat
- High-pull Olympic lifts
- Snatch (heavy)
- Clean (heavy)
- Sit-ups (heavy)
- GHD sit-ups (heavy)
- Overhead press (standing, heavy)

**Substitute with:**
- Trap bar deadlift (more upright torso)
- Sumo deadlift (more upright torso)
- Belt squat (no spinal load)
- Leg press (moderate)
- Chest-supported rows (takes stress off lower back)
- Lat pulldowns (seated, supported)
- Cable rows (seated)
- Machine rows (supported)
- Reverse lunges (lighter, controlled)
- Goblet squats (lighter)
- Planks (ab wheel rollouts if pain-free)
- Dead bugs
- Bird dogs
- Pallof press
- McGill curl-ups

**Modified options:**
- All pulling should be chest-supported or seated
- Avoid loaded spinal flexion/extension
- Use machines over free weights when possible
- Prioritize core stability work

## Wrist Injury / Wrist Pain (Carpal Tunnel, Tendonitis)

**Avoid these movements:**
- Front squat (rack position)
- Overhead press (barbell)
- Bench press (heavy, narrow grip)
- Clean (heavy)
- Snatch (heavy)
- Push-ups (if wrist extension is painful)
- Handstand push-ups
- Planks (on hands)
- Farmer's carries (heavy)

**Substitute with:**
- Safety squat bar (cushioned yoke)
- Trap bar (neutral grip)
- Dumbbell press (neutral grip)
- Machine press (neutral grip handles)
- Cable press (neutral grip)
- Push-ups on push-up bars or parallettes (neutral wrist)
- Planks on forearms
- Rower (lighter, supported)
- Lat pulldowns with neutral grip
- Use wrist wraps for heavy pressing
- Hammer curls (neutral grip)
- Chest-supported rows (neutral grip)

---

## Elbow Injury / Elbow Pain (Tennis Elbow, Golfer's Elbow)

**Avoid these movements:**
- Close-grip bench press
- Skull crushers (heavy)
- Tricep kickbacks (heavy)
- Wrist curls (heavy)
- Reverse wrist curls (heavy)
- Heavy gripping work

**Substitute with:**
- Overhead tricep extensions (lighter)
- Cable pushdowns (lighter, varied angles)
- Tricep dips (bodyweight, controlled)
- Hammer curls (neutral grip)
- Preacher curls (supported)
- Cable curls
- Use lifting straps to reduce grip demand
- Farmers carries with straps

---

## Hip Injury / Hip Pain (Labrum, Impingement)

**Avoid these movements:**
- Deep squat (below parallel)
- Sumo deadlift (wide stance)
- Bulgarian split squat (deep)
- Box jumps
- Running (high impact)
- Heavy hip thrust
- Hip adduction (heavy)

**Substitute with:**
- Belt squat (no hip flexion under load)
- Leg press (controlled ROM)
- Glute bridges (lighter)
- Single-leg press
- Step-ups (lower box)
- Incline treadmill walking
- Cycling (low resistance)
- Swimming
- Hip abduction work
- Clamshells
- Side-lying leg raises

---

## Ankle Injury / Ankle Mobility Issues

**Avoid these movements:**
- Olympic lifts (require ankle mobility)
- Deep squat (below parallel)
- Box jumps
- Plyometric work
- Running (if unstable)
- Lunges (deep)

**Substitute with:**
- Heel-elevated squats (2-5 lb plates under heels)
- Leg press (seated)
- Hack squat (guided path)
- Belt squat
- Cycling
- Swimming
- Upper body emphasis days
- Seated calf raises (no ankle stress)

---

## Pregnancy / Post-Partum (General Guidelines)

**Avoid these movements:**
- Heavy compound lifts
- Exercises lying flat on back (after first trimester)
- Heavy Valsalva maneuvers
- Jumping/plyometrics
- Contact risk exercises

**Substitute with:**
- Lighter resistance, higher reps
- Incline bench (slightly elevated)
- Supported seated exercises
- Cable work
- Walking
- Stationary cycling
- Swimming
- Pelvic floor work
- Diaphragmatic breathing exercises

**Recommendation:** Always defer to user's healthcare provider for specific modifications.

---

## Cardiovascular Limitations (Cannot Do High-Intensity)

**Avoid these movements:**
- Sprints (all forms)
- HIIT finishers
- Norwegian 4x4
- VO2 max protocols
- High-impact cardio
- Battle ropes (high intensity)

**Substitute with:**
- Zone 2 cardio only
- Incline treadmill walking
- Cycling (steady-state)
- Elliptical
- Swimming
- Rower (steady-state, moderate)
- Longer warm-ups, longer cooldowns

---

## Equipment-Specific Limitations (No Rack / No Heavy Weights)

**If no squat rack:**
- Substitute barbell back squats with goblet squats, belt squats, leg press, or hack squat
- Substitute barbell overhead press with dumbbell press or landmine press

**If no heavy weights:**
- Increase rep ranges (12-20 reps)
- Use tempo control (3-1-2 or 4-1-1) for time under tension
- Add pause reps
- Use bands for accommodating resistance
- Increase training frequency

**If limited equipment (home gym, bodyweight only):**
- Focus on tempo and volume
- Use unilateral movements
- Add bands/cables for variety
- Increase training frequency to 4-5 days
- Use advanced bodyweight progressions (pistol squats, archer push-ups, etc.)

## Parsing User Limitations (NLP Mapping)

When a user types free-text limitations, the AI should map common phrases:

| User Says | Maps To |
|---|---|
| "bad knees", "knee injury", "knee pain" | knee-injury |
| "shoulder impingement", "bad shoulder", "shoulder pain" | shoulder-injury |
| "lower back", "back pain", "disc issue", "sciatica" | lower-back-injury |
| "carpal tunnel", "wrist pain", "wrist injury" | wrist-injury |
| "tennis elbow", "elbow pain" | elbow-injury |
| "hip impingement", "hip pain", "labrum tear" | hip-injury |
| "bad ankle", "ankle mobility" | ankle-injury |
| "pregnant", "post-partum" | pregnancy |
| "can't do high intensity", "low intensity only" | cardio-limitation |
| "no box jumps", "no jumping" | plyo-limitation |
| "no overhead work" | overhead-limitation |
| "no deadlifts" | deadlift-limitation |

**Process:**
1. Parse user limitation text
2. Map to one or more limitation categories
3. Apply the EXCLUDE list to candidate movements
4. Apply the SUBSTITUTE list to fill in alternatives
5. Note the limitation in the program metadata for the user
